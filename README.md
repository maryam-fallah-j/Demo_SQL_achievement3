# Rockbuster Stealth Data Analysis Project
## Project Overview

Rockbuster Stealth LLC is a global movie rental company that previously operated physical stores around the world. Due to increasing competition 
from streaming services such as Netflix and Amazon Prime, Rockbuster’s management team decided to leverage its existing movie licenses to launch
an online video rental platform.
To support this strategic shift, the Rockbuster Stealth Management Board requested data-driven insights to inform their 2020 business strategy. 
This project analyzes Rockbuster’s operational database using SQL to answer their key business questions.

## Key Business Questions

+ Which movies contributed the most and least to overall revenue?

+ What was the average rental duration across all videos?

+ Which countries are Rockbuster customers located in?

+ Where are customers with high lifetime value (LTV) based?

+ Do sales figures vary across geographic regions?

## Tools Used

+ PostgreSQL – Open-source relational database used to store and manage Rockbuster data

+ pgAdmin 4 – Database management interface used to write and execute SQL queries

+ Tableau Public – Data visualization platform used to create dashboards and charts

+ PowerPoint – Used to present findings to stakeholders

## Data Overview

### The Rockbuster dataset contains information about the company’s operations, including films, customers, rentals, and payments.Key Tables Include:

+ Film Data – Movie titles, genres, languages, rental rates, and rental durations

+ Inventory Data – Information about film availability

+ Customer Data – Customer profiles, addresses, and geographic information

+ Rental Data – Rental transactions, rental dates, and return dates

+ Payment Data – Payment amounts and transaction dates

+ Store & Staff Data – Store locations and employee information

## Analytical Approach

+ Loaded the Rockbuster dataset into PostgreSQL

+ Explored relationships between tables using joins

+ Performed aggregations, subqueries, and filtering

+ Calculated revenue metrics and customer lifetime value

+ Identified geographic trends in customer distribution and sales

+ Visualized findings using Tableau

+ Delivered insights in a stakeholder-focused presentation

## Project Outcome

### This project provides actionable insights to help Rockbuster:

+ Identify top-performing and underperforming films

+ Understand customer geographic distribution

+ Detect high-value markets

+ Support strategic planning for online expansion
